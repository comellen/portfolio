Projects

Please check out my projects below! These projects were completed during my cohort at Eleven Fifty Academy.

Travall

Travall is a collaborative travel application built in Angular. Creating a Travall will allow you to invite others to your upcoming trip as well as display all transportation details and all activities you would like to do while on the trip.

berm

berm is a mountain biking app that allows the user to keep track of their bikes and components, trails, and trail times. This PERN (Postgres, Express, React, Node) app uses validation to display data unique to each user account.

Specialized Recipe Search

This page pulls data from the Edamam API based upon the user's search term, diet and health options, and food/recipe omissions. The data is then displayed on the page using JavaScript.

JavaScript Fundies

A static webpage describing JavaScript Fundamentals. This project was a practice in HTML and CSS styling. The inspiration for the style of this page comes from the color scheme Monokai Pro, the color scheme I most enjoy using in VS Code.

CSS Creature

This codePen is a simple exercise in CSS shaping and positioning. This was our first project at Eleven Fifty Academy.
